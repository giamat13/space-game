# 🚀 Space Defender: Skin System

ברוכים הבאים ל-**Space Defender**, משחק אקשן חללי מהיר וממכר המתרחש במעמקי הגלקסיה. בחרו את הספינה שלכם, התחמקו מאסטרואידים, השמידו אויבים ונסו להגיע לראש טבלת השיאים!

---

## ✨ מה מחכה לכם במשחק?

### 🛡️ צי חלליות (Skins)
אל תסתפקו בחללית אחת! ככל שתעלו ברמות, תוכלו לפתוח ספינות חזקות יותר:
* **Classic:** מאוזנת ומתאימה לכל משימה.
* **Interceptor:** מהירה עם קצב אש רצחני.
* **Tank:** איטית אך עמידה במיוחד עם הרבה נקודות חיים.
* **Phoenix (Level 3):** חללית אש עם יכולת ירי היקפית.
* **Vortex (Level 5):** חללית עוצמתית עם לייזר הרסני.
* **Joker (Level 10):** חללית מסתורית שמשנה את חוקי המשחק.

### ⚡ יכולות מיוחדות
לכל ספינה יש "קלף מנצח". טענו את האנרגיה והשתמשו ב:
* **Vortex Laser:** קרן עוצמתית שמוחקת כל מה שבדרך.
* **Phoenix Feathers:** מטח של נוצות אש לכל הכיוונים.
* **Joker Chaos:** שיבוש מערכות האויב והחלשתם זמנית.

### 🍔 מכניקת הבונוסים הייחודית
אספו רכיבי המבורגר (לחמנייה, קציצה וירקות) כדי להרכיב המבורגר שלם!
* **בונוס:** המבורגר מלא מעניק ריפוי ונקודות רבות.
* **אזהרה:** אם תאכלו יותר מדי כשאתם בריאים, תהפכו ל-"Fat Player" ותהיו איטיים יותר עד שתספגו נזק ותחזרו לכושר.

---

## 🕹️ איך משחקים?

השליטה נוחה ומותאמת אישית:

* **תנועה:** הזזת העכבר מצד לצד (החללית עוקבת אחרי הסמן) או חצי מקלדת
* **ירי:** מקש `Space` (רווח) או לחיצת עכבר
* **יכולת מיוחדת:** מקש `B` או קליק ימני בעכבר
* **השהיה/תפריט:** מקש `Esc`

> [!NOTE]
> ניתן לשנות את מקשי השליטה בכל עת דרך תפריט ה-**Settings** במשחק.

## 🏆 טיפים לטייסים מתחילים
1. **אויבים כתומים (Elite):** הם קשים יותר, אבל השמדתם מעניקה ריפוי משמעותי (Repair).
2. **אסטרואידים:** הם לא יורים, אבל הם דורשים הרבה פגיעות כדי להישבר. אל תתנו להם להתנגש בכם!
3. **טבלת שיאים:** המשחק שומר את 5 התוצאות הטובות ביותר שלכם. האם תצליחו לעקוף את השיא של עצמכם?

## 🔧 התיקון שבוצע

הבעיה המקורית הייתה שהמשחק לא טען כראוי בגלל בעיות עם:
- סדר טעינת הסקריפטים
- שימוש ב-`onclick` attributes במקום event listeners
- בעיות ייצוא/ייבוא של modules

**התיקונים:**
✅ כל הסקריפטים נטענים כ-ES6 modules בסדר הנכון  
✅ כל הכפתורים משתמשים ב-event listeners במקום onclick  
✅ ניקוי והסרה של קוד מיותר  
✅ תיקון כל בעיות ה-import/export

## 📁 קבצי המשחק

* **index.html** - קובץ ה-HTML הראשי
* **styles.css** - כל הסגנונות והאנימציות
* **data.js** - נתוני המשחק, סקינים, ניהול cookies
* **systems.js** - מערכות השחקן, ירי, אפקטים ויצירת אויבים
* **updates.js** - לולאת העדכון הראשית - תנועת כדורים, אויבים, התנגשויות
* **main.js** - קובץ הראשי שמחבר הכל ומטפל באירועים

## 🎮 פקודות debug

פתח את console והקלד:
```javascript
debugUnlockAllSkins()       // פתח את כל הסקינים
debugUnlockSkin('phoenix')  // פתח סקין ספציפי
debugListSkins()            // הצג רשימת סקינים
setLvl(10)                  // קפוץ לרמה 10 (המשחק חייב לרוץ)
spawn('elite')              // צור אויב מסוג elite
```
